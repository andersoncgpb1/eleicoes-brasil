Coloque a apuração das eleições 2014 no seu site em tempo real.

== Description ==

Esse plugin mostra em tempo real os dados da apuração 2014 em seu site, direto do site do TSE.

== Installation ==

1. Faça upload do plugin `Apuração Eleições Brasil` no diretório `/wp-content/plugins/`
1. Ative o plugin no menu 'Plugins' 
1. Inclua o shortcode [apuracao] em qualquer página ou post.
